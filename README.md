## **Домашнее задание 20.2 Шаблонизация**

1. Созданы новый контроллер и шаблон, которые будут отвечать за 
отображение отдельной страницы с товаром, 
на которой необходимо вывести всю информацию о самом товаре. 

2. В созданный ранее шаблон для главной страницы выведен список товаров в цикле. 
Для единообразия выводимых карточек отображаемое описание обрезано 
после первых выведенных 100 символов. 

3. Выделен общий (базовый) шаблон, а также подшаблон с главным меню.

4. Реализован шаблонный фильтр, который преобразует переданный путь 
в полный путь для доступа к медиафайлу.
