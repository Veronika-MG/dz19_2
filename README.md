## **Домашнее задание 21.1 FBV и CBV**

1. Переведены имеющиеся контроллеры с FBV на CBV.

2. Создана новая модель блоговой записи со следующими полями:
-заголовок;
-slug (реализовать через CharField);
-содержимое;
-превью (изображение);
-дата создания;
-признак публикации;
-количество просмотров.
Для работы с блогом реализован CRUD для новой модели. 

3. Модифицирован вывод и обработка запросов, добавлена следующая логика 
на уровне контроллеров:
-При открытии отдельной статьи увеличивается счетчик просмотров.
-Выводятся в список статей только те, которые имеют положительный 
признак публикации.
-При создании динамически формируется slug name для заголовка.
-После успешного редактирования записи перенаправляется 
пользователь на просмотр этой статьи.
